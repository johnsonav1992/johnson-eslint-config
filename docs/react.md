# React

## Rules

- **react/prop-types**: Disable prop type checking.
- **react/jsx-tag-spacing**: Disallow inconsistent spacing around JSX brackets.
- **react/jsx-curly-spacing**: Enforce consistent spacing around JSX attributes.
- **react/jsx-max-props-per-line**: Enforce all props are on a new line.
- **react/jsx-first-prop-new-line**: Enforce first prop on a new line if the component has multiple props.
- **react/self-closing-comp**: Enforce self closing tags for React and HTML elements.
- **react/jsx-wrap-multilines**: Enforce parentheses around multilines JSX.
- **react/jsx-closing-bracket-location**: Enforce consistent alignment of closing brackets of JSX elements.
- **react/jsx-boolean-value**: Enforce boolean attributes to be explicit.
- **react/jsx-curly-brace-presence**: Disallow curly brackets as props or children in JSX expressions.
